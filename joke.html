Joke API

For this project, I was able to learn how to use a web API to get a random joke and display it in two parts: the set up and the punch line. There is a brief pause before it displays the punchline. The code makes an HTTP request to the jokes API and uses requests library to print the set up and then it is followed by a 5-second delay using time.sleep() before it displays the punchline.
